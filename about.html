Tentang Saya

Tentang Saya

Nama Saya Stefanus, biasa dipanggil Evan. Kuliah di Universitas MaChung, jurusan Sistem Informasi tahun 2018.

Tujuan Saya

Tujuan dari adanya website ini adalah untuk mengenalkan Batik khas Indonesia kepada khalayak luar sana.

Rencana Ke Depan

Semoga dengan adanya website ini lebih banyak masyarakat yang mengerti akan Batik asal Indonesia ini. Dan saya akan terus mengembangkan website ini sesuai dengan kebutuhannya.

Apa saja dalam website?

Website Batik Indonesia dengan API Batikita

Dalam website ini berisikan pengenalan batik yang ada di Indonesia beserta nama, makna dari batik, asal, hingga harga batik.

Menyediakan Batik dari beberapa daerah

Di mana website ini mencakup Batik yang ada di daerah yang ada di Negara Indonesia.

List Data Batik

Di mana pada website ini telah ada list Batik baik itu daerah batik, makna batik, hingga harga batik tersebut.
